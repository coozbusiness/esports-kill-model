# CHROME EXTENSION — INSTALLATION GUIDE

Takes 2 minutes. No Chrome Web Store needed.

═══════════════════════════════════════════════
STEP 1 — LOAD THE EXTENSION
═══════════════════════════════════════════════

1. Open Chrome
2. Go to: chrome://extensions
3. Turn ON "Developer mode" (toggle in top-right corner)
4. Click "Load unpacked"
5. Navigate to your esports-app folder → select the "chrome-extension" folder
6. Click "Select Folder"

You should see "Esports Kill Model — PrizePicks Auto-Import" appear in your extensions list.

═══════════════════════════════════════════════
STEP 2 — SET YOUR APP URL
═══════════════════════════════════════════════

1. Click the extension icon in Chrome toolbar (puzzle piece icon → pin it)
2. In the popup, find "YOUR APP URL"
3. Make sure it shows your Vercel URL (e.g. https://esports-kill-model.vercel.app)
4. Click SAVE URL

═══════════════════════════════════════════════
DAILY USAGE
═══════════════════════════════════════════════

1. Click extension icon → click "OPEN PRIZEPICKS"
2. Browse esports props normally — scroll through them
3. Extension captures the data automatically in the background
4. A green indicator appears at bottom-right of PrizePicks page
5. Click extension icon → click "★ SEND TO KILL MODEL"
6. Your app opens with all props loaded — no copy/paste ever

═══════════════════════════════════════════════
TROUBLESHOOTING
═══════════════════════════════════════════════

"No data — browse PrizePicks first"
→ You need to browse the props page first so PrizePicks loads the API data
→ Go to app.prizepicks.com → click on Esports category → wait 2 seconds

Extension not appearing in toolbar:
→ Go to chrome://extensions and make sure it's enabled
→ Click the puzzle piece in Chrome toolbar and pin the extension

Props load but wrong ones appear:
→ Tell the developer (me) what sport/league was missing

Need to update the extension after code changes:
→ Go to chrome://extensions → find the extension → click the refresh icon (↺)
